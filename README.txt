≪起動方法≫
「Grandioso.exe」をダブルクリックして実行してください。

≪操作方法≫
【キーボード】
移動：WASD, 矢印キー
決定：Enter
キャンセル：Escape

【コントローラー】
移動：十字キー, 方向スティック
決定：A
キャンセル：B
